# 性能標準（Performance Standards）

* Version: 0.1.0（Proposed / ドラフト）
* Date: 2026-04-01
* 上位規範: constitution.md（開発憲章「9. 完了条件」）

本書は、憲章が参照する性能目標・測定基準（SLO・性能予算）の正本（SSoT）です。憲章と矛盾する場合は憲章が優先します（MUST）。

---

## 1. 性能目標（SLO・性能予算）

* API は既定で p95 レイテンシ < 1s を目標とし、機能ごとに spec の非機能要求で上書きします（SHOULD）。
* 重要経路にはエラーバジェットを定め、超過時の対応方針を持つべきです（SHOULD）。

---

## 2. 性能テストと回帰防止

* 変更は重大な性能劣化を導入してはなりません（憲章「9. 完了条件」）。
* 重要経路は負荷試験・ベンチマークで継続的に確認するべきです（SHOULD）。測定基準は本書を正本とします。
